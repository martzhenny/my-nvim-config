# My Neovim Config

## Overview
This repository contains my personal Neovim configuration files. It is designed to improve my coding workflow and provide a customized editing experience.

## Features
- Syntax highlighting
- Code completion
- Custom key mappings

## Installation
To use this configuration, clone the repository and copy the files to your Neovim configuration directory:

```bash
git clone https://github.com/martzhenny/my-nvim-config.git
cp -r my-nvim-config/* ~/.config/nvim/
```

Feel free to modify any settings to fit your workflow!
